問題</strong>

Refactor > Rename でメソッドの名前を変更するときに、いつもSnapshotが生成されていました。このSnapshot機能は、プロジェクトの状態を保存するものですが、ほとんど利用しません。毎回、Shapshotの生成を問われるのが面倒なので、この機能をオフにします。

環境</strong>

Xcode version: 4.2 Build 4C199

Mac OS: 10.6.8

方法</strong>

  1. Files > Project Settingsを開く。</a></li>
  2. Snapshotタブを選択し、"Create snapshot of project before mass-editing operations"のチェックボックスのチェックを外す。</a></li>
    </ol>
