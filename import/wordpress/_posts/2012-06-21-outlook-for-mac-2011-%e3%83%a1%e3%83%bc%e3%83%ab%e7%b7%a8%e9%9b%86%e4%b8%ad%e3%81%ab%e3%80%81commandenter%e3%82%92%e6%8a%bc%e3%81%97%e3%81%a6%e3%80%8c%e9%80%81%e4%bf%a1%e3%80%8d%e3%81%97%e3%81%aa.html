問題</strong>

</strong>Outlook for Mac 2011を仕事で使っているときに、メール編集中にCommand+Enterを押してしまい、下書き状態のメールを送信してしまうトラブルが多々発生して困っていました。

解決方法</strong>

  1. System Preference > Hardware > Keyboard を開く</li>
  2. Keyboard Shortcutの左ペインの一番下にある、"Application Shortcuts"を選択</li>
  3. 右ペイン下の"+"ボタンを押す</li>
  4. Applicationで、Outllook for Macを選択</li>
  5. Menu Titleの"Send"(または、"Send")を入力（日本語版の場合、"送信”となります)</li>
  6. Keyboard Shourtcutで、^⌘↩と割り当てる</li>
  7. Outllook for Macを開いている場合は、アプリケーションを再起動</li>
  8. 新規メール画面で、適当なアドレスを入力して、⌘↩と押して確認</li>
    </ol>
