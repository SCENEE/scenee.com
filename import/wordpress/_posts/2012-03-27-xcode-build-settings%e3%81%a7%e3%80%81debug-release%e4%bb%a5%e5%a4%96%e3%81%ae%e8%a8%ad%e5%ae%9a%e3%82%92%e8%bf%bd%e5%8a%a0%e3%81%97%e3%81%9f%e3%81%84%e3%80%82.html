問題</strong>

Xcodeでのあるプロジェクトにおいて、Ad HocやAppStore用のschemaを作成するためにBuild SettingsにDebug,ReleaseのBuild Configuration以外に、Ad HocやAppStoreというBuild Configurationを追加したい

解決方法</strong>

  1. Project Navigatorで、プロジェクトファイルを選択する</li>
  2. 右ペインで、Projectファイルのアイコンを選択する</li>
  3. infoタブを開く</li>
  4. そこに表示されているConfigurationグループのフッターにある+ボタンでBuild Configurationを追加する。</li>
