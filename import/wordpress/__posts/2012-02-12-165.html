問題</strong>

Xcode4.2でビルドをしたときに以下のような警告が出た。

[WARN]warning: no rule to process file '$(PROJECT_DIR)/######' of type sourcecode.c.h</pre>

解決方法</strong>


これは、ビルドしたターゲットの Build Phases に.hファイルが含まれているからでした。.hファイルを削除したら、警告は出なくなりました。
