Problem:</strong>

Amazon EC2でWindows Server 2008を利用していると、頻繁にRemoteDesktopConnectionを使います。

通常は、問題はないのですが、稀に、リモートログイン時、またはリモートログイン中に、画面上に何も表示されなくなる(ログイン画面のバックグラウンドカラー一色になる)ことがあります。

Solution:</strong>

  1. Ctrl + Alt + Endを押します。</li>
  2. ログアウトを選択する。</li>
    </ol>
